🖥️ Windows Updates

Updates

Check, install, manage, and troubleshoot Windows Updates. Includes cache clearing and stuck update fixes.

Check & Install Updates

#What to DoHow to Do It
1Check for updatesGo to Start > Settings > Windows Update > Check for Updates. Let it finish.
2Restart to finish updatesSave all your work first, then go to Start > Power > Update and Restart.
3View update historyGo to Settings > Windows Update > Update History to see what has been installed.

Manage Updates

#What to DoHow to Do It
1Pause updates temporarilyGo to Settings > Windows Update > Pause Updates. Choose up to 5 weeks.

Clear Windows Update Cache

#What to DoHow to Do It
1Stop Windows Update serviceOpen Services, find Windows Update, right-click > Stop.
2Delete cached update filesDelete all files in C:\Windows\SoftwareDistribution\Download\.
3Restart Windows Update serviceGo back to Services, right-click Windows Update > Start. Then check for updates again.

Troubleshoot Updates

#ProblemFix
1Update is stuckRestart your computer. If still stuck, go to Settings > System > Troubleshoot > Other Troubleshooters > Windows Update > Run.
2Updates failed to installRun the Windows Update Troubleshooter (above), then try clearing the cache (see section above).

Quick Reference — One-Liners
