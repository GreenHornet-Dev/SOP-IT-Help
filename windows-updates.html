🖥️ Windows Updates

Updates

Keep Windows up to date and fix stuck updates.

Steps

#What to DoHow to Do It
1Check for updatesGo to Start > Settings > Windows Update > Check for Updates. Let it finish.
2Update is stuckRestart your computer. If still stuck, go to Windows Update > Troubleshoot > Windows Update and click Run.
3Pause updates temporarilyGo to Start > Settings > Windows Update > Pause Updates. Choose up to 5 weeks.
4Update historyGo to Settings > Windows Update > Update History to see what has been installed.
5Clear update cacheOpen Services, find Windows Update, right-click > Stop. Then delete files in C:\Windows\SoftwareDistribution\Download. Restart the service.
6Updates failed to installRun Windows Update Troubleshooter from Settings > System > Troubleshoot > Other Troubleshooters.
7Restart to finish updatesSave all your work first, then go to Start > Power > Update and Restart.
