💾 Drive Management

Tag: Drives

Manage storage drives and network folders.

Steps

# What to do How to do it
1 Map a network drive Open File Explorer > This PC > Map network drive. Enter the folder path and check Reconnect at sign-in.
2 Check disk space Open File Explorer and look at your drives. Right-click a drive > Properties to see used/free space.
3 Run disk cleanup Search for Disk Cleanup in Start. Select drive C:. Check items to delete and click OK.
4 Fix drive not showing Open Disk Management (search in Start). Right-click the drive and assign a letter.
5 Disconnect network drive Open File Explorer > This PC. Right-click the mapped drive > Disconnect.
