💾 Drive Management

Drives

Map and disconnect network drives, check disk space, and fix drives that aren't showing.

Map & Disconnect Network Drives

#What to DoHow to Do It
1Map a network driveOpen File Explorer > This PC > Map network drive. Enter the folder path and check Reconnect at sign-in.
2Disconnect network driveOpen File Explorer > This PC. Right-click the mapped drive > Disconnect.

Disk Space & Cleanup

#What to DoHow to Do It
1Check disk spaceOpen File Explorer and look at your drives. Right-click a drive > Properties to see used/free space.
2Run disk cleanupSearch for Disk Cleanup in Start. Select drive C:. Check items to delete and click OK.

Troubleshooting

#ProblemFix
1Drive not showing in File ExplorerOpen Disk Management (search in Start). Right-click the drive and assign a drive letter.
2Network drive disconnects on rebootRe-map the drive and check Reconnect at sign-in. Or use the persistent net use one-liner below.

Quick Reference — One-Liners

TaskCommand (PowerShell)
Map a network drive (persistent)net use Z: \\server\share /persistent:yes
Disconnect all network drivesnet use * /delete /yes
Check disk space (all drives)Get-PSDrive -PSProvider FileSystem | Select-Object Name, @{N="Used(GB)";E={[math]::Round($_.Used/1GB,1)}}, @{N="Free(GB)";E={[math]::Round($_.Free/1GB,1)}}
List mapped drivesGet-SmbMapping | Select-Object LocalPath, RemotePath, Status
Run disk cleanup silentlycleanmgr /sagerun:1
Open Disk ManagementStart-Process diskmgmt.msc
