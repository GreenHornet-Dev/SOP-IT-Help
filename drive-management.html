💾 Drive Management

Drives

Manage storage drives and network folders.

Steps

#What to DoHow to Do It
1Map a network driveOpen File Explorer > This PC > Map network drive. Enter the folder path and check Reconnect at sign-in.
2Check disk spaceOpen File Explorer and look at your drives. Right-click a drive > Properties to see used/free space.
3Run disk cleanupSearch for Disk Cleanup in Start. Select drive C:. Check items to delete and click OK.
4Fix drive not showingOpen Disk Management (search in Start). Right-click the drive and assign a letter.
5Disconnect network driveOpen File Explorer > This PC. Right-click the mapped drive > Disconnect.
