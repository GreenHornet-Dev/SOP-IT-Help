🌐 Web Browser Management

Browser

Clear cache, manage extensions, reset browser settings, and set the default browser.

Clear Cache & Cookies

#What to DoHow to Do It
1Clear browser cache (UI)Go to Settings > Privacy > Clear browsing data. Check Cache and Cookies. Click Clear data.
2Clear cache — keyboard shortcutPress Ctrl+Shift+Delete in any browser to open the clear data dialog directly.

Manage Browser Settings

#What to DoHow to Do It
1Update the browserGo to Settings > Help > About. The browser will check for and apply updates automatically.
2Reset browser settingsGo to Settings > Reset settings. This fixes most unusual browser behavior without losing bookmarks.
3Manage extensionsGo to Settings > Extensions. Disable or remove extensions causing issues.
4Set default browserGo to Windows Settings > Apps > Default Apps. Choose your preferred browser.

Quick Reference — One-Liners

TaskCommand (PowerShell)
Clear Edge cacheRemove-Item -Path "$env:LOCALAPPDATA\Microsoft\Edge\User Data\Default\Cache\*" -Recurse -Force -ErrorAction SilentlyContinue
Clear Chrome cacheRemove-Item -Path "$env:LOCALAPPDATA\Google\Chrome\User Data\Default\Cache\*" -Recurse -Force -ErrorAction SilentlyContinue
Kill EdgeStop-Process -Name msedge -Force -ErrorAction SilentlyContinue
Kill ChromeStop-Process -Name chrome -Force -ErrorAction SilentlyContinue
Set Edge as default (registry)Start-Process "ms-settings:defaultapps"
