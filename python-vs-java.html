1. Quick Decision Guide

If you only read one section — start here.

Bottom line for IT staff: Learn Python first. It solves real problems fast, every IT tool has a Python SDK, and the entire AI ecosystem runs on it.

3. Java — Strengths & When It Matters

Where Java Genuinely Wins

StrengthWhat it means in practice
Performance at scaleHandles millions of requests per second — banks, airlines, enterprise ERP
Strong type systemCompiler catches errors before runtime — safer in large teams
Android developmentNative Android apps are Java (or Kotlin, which runs on the same JVM)
Enterprise ecosystemSpring Boot, Hibernate, Maven — huge in corporate backend systems
Long-running stabilityJava 8 code written in 2014 still runs in production today — extreme backwards compatibility

The Honest Downside for IT Starters

FrictionWhat it feels like
Verbose boilerplate10 lines of setup before you do anything useful — discouraging for quick scripts
Compile step requiredCan't just run a file — must compile first (slower feedback loop)
Late to AI/MLAI libraries exist but are secondary — Python ecosystem is 5 years ahead
Overkill for scriptsA 5-line Python script becomes a full project structure in Java

Learn Java when: A specific job posting requires it, or you're building Android apps. Otherwise, Python gets you productive faster on everything IT-related.

Same Task — Different Languages

Fetch data from a REST API and print a field:

# Python — 4 lines
import requests
r = requests.get('https://api.example.com/asset/PC-001', headers={'Authorization': 'Bearer TOKEN'})
print(r.json()['name'])
// Java — same result, much more setup
import java.net.http.*;
import java.net.URI;
HttpClient client = HttpClient.newHttpClient();
HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create("https://api.example.com/asset/PC-001"))
    .header("Authorization", "Bearer TOKEN")
    .build();
HttpResponse<String> response = client.send(request, HttpResponse.BodyHandlers.ofString());
System.out.println(response.body()); // still need to parse JSON separately

5. Python for IT Automation — Real Examples

These are the kinds of scripts you'd actually run in the GreenHornet / Newsbank environment.

Query Lansweeper GraphQL API

import requests

url = "https://api.lansweeper.com/api/v2/graphql"
token = "your-PAT-here"
query = """
{
  Site {
    site {
      assetBasicInfo(where: { assetBasicInfo: { name: { like: "%PC-001%" } } }) {
        name ipAddress userName lastSeen
      }
    }
  }
}
"""
r = requests.post(url, json={"query": query}, headers={"Authorization": f"Bearer {token}"})
for asset in r.json()["data"]["Site"]["site"]["assetBasicInfo"]:
    print(asset["name"], asset["ipAddress"], asset["userName"])

Fetch Open Jira Tickets

import requests, base64

jira_url = "https://newsbank.atlassian.net"
email = "you@newsbank.com"
token = "your-api-token"
auth = base64.b64encode(f"{email}:{token}".encode()).decode()

r = requests.get(
    f"{jira_url}/rest/api/3/search",
    params={"jql": "assignee = currentUser() AND resolution = Unresolved", "maxResults": 10},
    headers={"Authorization": f"Basic {auth}", "Accept": "application/json"}
)
for issue in r.json()["issues"]:
    print(issue["key"], issue["fields"]["summary"])

Export Lansweeper SQL Report to CSV

import pyodbc, csv

conn = pyodbc.connect("DRIVER={SQL Server};SERVER=your-server;DATABASE=lansweeperdb;Trusted_Connection=yes")
cursor = conn.cursor()
cursor.execute("SELECT a.Name, ac.Username, a.IPAddress FROM tblAssets a JOIN tblAssetCustom ac ON a.AssetID = ac.AssetID")

with open("assets.csv", "w", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["Name", "Username", "IP"])
    writer.writerows(cursor.fetchall())

print("Exported assets.csv")

POST to a Power Automate Webhook

import requests

webhook_url = "https://prod-xx.westus.logic.azure.com:443/workflows/..."
payload = {"alert": "Disk space critical", "host": "PC-FILE-01", "threshold": "90%"}
r = requests.post(webhook_url, json=payload)
print("Flow triggered:", r.status_code)

Call the Anthropic (Claude) API

import anthropic

client = anthropic.Anthropic(api_key="your-key")
message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Summarize this IT incident report: ..."}]
)
print(message.content[0].text)

7. Getting Started — Install & First Script

Install Python (Windows)

StepAction
1Install via winget: winget install Python.Python.3.12
2Verify: open PowerShell → python --version
3Upgrade pip: python -m pip install --upgrade pip
4Install VS Code Python extension (if not already): code --install-extension ms-python.python

Your First IT Script — 60 Seconds

Create a file called hello_it.py and paste this:

import requests

# Test: call a public API and print a result
r = requests.get("https://httpbin.org/json")
data = r.json()
print("API call worked:", data["slideshow"]["title"])

Run it: python hello_it.py

That's the pattern for every API you'll ever call — request → parse JSON → use the data. Lansweeper, Jira, Microsoft Graph — all the same shape.

Virtual Environments (Keep Projects Isolated)

# Create a virtual env for a project
python -m venv .venv

# Activate it (PowerShell)
.\.venv\Scripts\Activate.ps1

# Now install libraries — they stay inside this project only
pip install requests anthropic pandas

Always use a virtual env per project. Keeps library versions from conflicting between scripts.

Secrets — Never Hardcode Keys

Create a .env file in your project folder:

JIRA_TOKEN=your-token-here
LS_PAT=your-pat-here
ANTHROPIC_KEY=your-key-here

Load it in your script:

from dotenv import load_dotenv
import os

load_dotenv()
token = os.getenv("JIRA_TOKEN")

Add .env to your .gitignore — same redacted build pattern as Clippy's localStorage. Keys stay local, code stays clean.
