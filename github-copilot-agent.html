🤖 GitHub Copilot Agent (Browser)

AI Tools

Use GitHub Copilot coding agent directly from your browser to automate code changes and create pull requests.

Using the Copilot Agent

#What to DoHow to Do It
1Open GitHub repoNavigate to your repository on github.com and sign in with your GitHub account.
2Open Copilot ChatClick the Copilot icon (✨) in the top nav bar or press Ctrl+K to open GitHub Copilot Chat.
3Describe your taskType a clear description of what you want Copilot to build or fix. Be specific — include file names and expected behavior.
4Assign to Copilot AgentIn the chat, click Assign to Copilot or use the issue sidebar to assign Copilot as the agent. Copilot will create a branch and PR automatically.
5Review the Pull RequestGo to Pull Requests, find the Copilot-generated PR, review the diff, and leave comments if changes are needed.
6Merge or request changesIf the code looks good, click Merge pull request. If not, add review comments and Copilot will iterate.
7Check CI/CD statusAfter merging, verify the GitHub Actions workflows pass (green checkmarks) under the Actions tab.
