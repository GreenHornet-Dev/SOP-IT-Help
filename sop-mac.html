macOS Terminal

Mac One-Liners

Quick terminal commands for help desk techs — run via LogMeIn Resolve remote session or Recovery Mode. Click any row to copy.

LogMeIn Resolve — Terminal Session

All commands run in macOS Terminal via LogMeIn Resolve remote session. Boot-failure commands run from Recovery Mode → Utilities → Terminal. Recovery Mode: hold Cmd+R (Intel) or Power (Apple Silicon).

Task Command Copy
Network
Show IP addressesen0 = Wi-Fi / en1 = Ethernet ipconfig getifaddr en0; ipconfig getifaddr en1
Flush DNS cache sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder
Show Wi-Fi network name networksetup -getairportnetwork en0
Release / renew DHCP sudo ipconfig set en0 DHCP
Ping test (4 packets) ping -c 4 8.8.8.8
DNS resolution test nslookup google.com
System Info
OS version + serial number sw_vers; system_profiler SPHardwareDataType | grep -E "Serial|Model"
Hostname hostname
Uptime uptime
Disk space df -h /
RAM usage vm_stat | head -5
MAC address (en0) ifconfig en0 | grep ether
User Account
Current user whoami; id
List all local users dscl . list /Users | grep -v '^_'
Check admin group members dscl . read /Groups/admin GroupMembership
Reset passwordReplace USERNAME and newpassword sudo dscl . -passwd /Users/USERNAME newpassword
Fix home folder permissions sudo chown -R $(whoami) ~/
Software / App Issues
Kill frozen app by name pkill -x "App Name"
Force-quit all instances killall "App Name"
Clear app cacheReplace com.vendor.AppName rm -rf ~/Library/Caches/com.vendor.AppName
List recent crash logs ls -lt ~/Library/Logs/DiagnosticReports | head -50
Rebuild Spotlight index sudo mdutil -E /
Printer
List printers lpstat -p
Clear print queueReplace PRINTERNAME cancel -a PRINTERNAME
Remove printer (reset)Replace PRINTERNAME sudo lpadmin -x PRINTERNAME
LogMeIn / Remote
Check LogMeIn agent running ps aux | grep -i logmein
Restart LogMeIn service sudo launchctl stop com.logmein.logmeindaemon && sudo launchctl start com.logmein.logmeindaemon
Check SIP status csrutil status
List startup items ls /Library/LaunchDaemons/ /Library/LaunchAgents/
NVRAM / Boot Flags
Flush NVRAM (Intel only)Reboot required — not needed on Apple Silicon sudo nvram -c
Force safe mode next boot sudo nvram boot-args="-x"
Clear safe mode flag sudo nvram boot-args=""
Check safe mode flags nvram boot-args | grep "-x" && echo "Safe Mode set" || echo "Normal boot"
Boot Failure — Recovery Mode Terminal
List all disks and volumes diskutil list
Repair main disk diskutil repairDisk disk0
Repair boot volume diskutil repairVolume disk1s1
APFS container repair diskutil apfs repairContainer disk0
fsck filesystem check (APFS) fsck_apfs -y /dev/disk1s1
Reset startup disk bless --mount "/Volumes/Macintosh HD" --setBoot --verbose
Last boot log (tail 100) log show --last boot | tail -100
List panic logs ls /Library/Logs/DiagnosticReports/ | grep panic
Mount read-write (Single User — Intel) /sbin/fsck -fy; /sbin/mount -uw /
Check FileVault status fdesetup status
Disable SIP (Recovery only — re-enable after) csrutil disable
Lansweeper Mac Agent (lsAgent)
Find lsagent.ini location sudo find /Library -name "lsagent.ini" 2>/dev/null
Stop lsAgent service sudo launchctl stop com.lansweeper.lsagent
Clear lsagent.ini (full reset) sudo rm /Library/LsAgent/lsagent.ini
Clear UniqueID only (re-scan, no re-register) sudo sed -i '' 's/^UniqueID=.*/UniqueID=/' /Library/LsAgent/lsagent.ini
Restart lsAgent sudo launchctl start com.lansweeper.lsagent
One-liner: stop → clear → restart sudo launchctl stop com.lansweeper.lsagent; sudo rm /Library/LsAgent/lsagent.ini; sudo launchctl start com.lansweeper.lsagent
Watch lsAgent log sudo tail -f /Library/LsAgent/lsagent.log
