# 🔄 Windows Updates

**Tag:** Updates

Keep Windows up to date and fix stuck updates.

## Steps

| # | What to do | How to do it |
|---|------------|--------------|
| 1 | Check for updates | Go to Start > Settings > Windows Update > Check for Updates. Let it finish. |
| 2 | Update is stuck | Restart your computer. If still stuck, go to Windows Update > Troubleshoot > Windows Update and click Run. |
| 3 | Pause updates temporarily | Go to Start > Settings > Windows Update > Pause Updates. Choose up to 5 weeks. |
| 4 | Update history | Go to Settings > Windows Update > Update History to see what has been installed. |
| 5 | Clear update cache | Open Start, type Services, find Windows Update, right-click > Stop. Then delete files in C:\\Windows\\SoftwareDistribution\\Download. Restart the service. |
| 6 | Updates failed to install | Run Windows Update Troubleshooter from Settings > System > Troubleshoot > Other Troubleshooters. |
| 7 | Restart to finish update | Save all your work first, then go to Start > Power > Update and Restart. |
