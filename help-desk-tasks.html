🎟️ Common Help Desk Tasks

Help Desk

Routine IT help desk tasks and resolutions.

Steps

#What to DoHow to Do It
1Password resetGo to Okta or the company login portal and click Forgot Password. Follow the steps to reset via email.
2Remote desktop assistanceUse LogMeIn Resolve or the assigned remote tool to connect to the user's machine. Get their permission first.
3New hardware requestSubmit a ticket in the help desk system with the user's name, department, and hardware needed.
4Software installationCheck if the software is approved. Deploy via Lansweeper or RMM tool, or guide the user through manual install.
5Account lockoutCheck Active Directory or Okta for the locked account. Unlock it and confirm with the user.
6Slow computerCheck Task Manager for high CPU/RAM usage. Restart, clear temp files, and run Windows Update.
