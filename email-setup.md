# 📧 Email / Mail Setup & Troubleshooting

**Tag:** Mail

Set up and fix email problems.

## Steps

| # | What to do | How to do it |
|---|------------|--------------|
| 1 | Add email account in Outlook | Open Outlook > File > Add Account. Enter your email and follow the prompts. |
| 2 | Fix emails not sending | Check your internet connection. Then go to Outbox and try resending. If stuck, restart Outlook. |
| 3 | Recover deleted emails | Go to the Deleted Items folder. Right-click email > Restore. |
| 4 | Set up Out of Office | File > Automatic Replies. Set the date range and message. |
| 5 | Fix Outlook not opening | Close Outlook, open Task Manager and end any Outlook processes, then reopen. |
